# 🚀 Task — Website Ideation & Git Practice

## Objective
To practice Git and GitHub collaboration.

---

## Task Overview

Think of a website idea that solves a real problem or fills a gap.

You will start by planning the website and focusing on one webpage from it.

---

## Step 1 — Website Ideation

Think of a website idea such as:
- A website for a business idea  
- A platform solving a real-world problem  
- A website that fills a gap or improves an existing solution  

---

## Step 2 — Create Your Ideation File

Create a `.txt` file on your system and write your idea clearly in the following format:

Website Title:
Problem:
Idea:
Features:
First Webpage Plan:

Keep it simple — explain:
- What your website is about  
- What problem it solves  
- What your first webpage will contain  

---

## Step 3 — Git Workflow (Important)

You MUST follow these steps:

1. Clone this repository  
2. Create your file (`yourname.txt`)  
3. Add your ideation  
4. Push your file to this repository  

---

## Important Notes

- Do NOT write directly on GitHub  
- Work locally using Git  
- Keep your idea simple and clear  

---

## Final Note

Think simple, think practical 💡  
This is your first step towards building a real website.
